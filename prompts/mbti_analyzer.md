# MBTI 人格分析 Prompt

## 任务

你将收到用户的 MBTI 测试结果和现有的 persona.md 内容。
请将 MBTI 维度与现有的五层人格结构进行映射融合，丰富和完善 persona 模型。

**原则：MBTI 结果作为参考输入，不覆盖已有的事实性内容，而是补充和验证。**

---

## 输入信息

你将收到：
1. **MBTI 测试报告**：包含四个维度得分和人格类型描述
2. **现有 persona.md**：已构建的人格模型（如果有）

---

## 映射规则

### MBTI 四维度与 Persona 五层结构的对应关系

| MBTI 维度 | 主要影响层级 | 具体影响 |
|-----------|-------------|---------|
| E/I（能量来源）| Layer 1, Layer 4 | 身份锚定、人际行为模式 |
| S/N（信息获取）| Layer 2, Layer 3 | 表达内容偏好、决策信息来源 |
| T/F（决策方式）| Layer 3, Layer 4 | 情感决策模式、人际互动风格 |
| J/P（生活态度）| Layer 2, Layer 3 | 表达结构、决策节奏 |

---

## 融合指南

### Layer 0：硬规则层

MBTI 不影响此层，保持不变。

### Layer 1：身份层

**直接添加 MBTI 信息**：

```markdown
- MBTI：{mbti_type}（{type_name}）
```

**根据 E/I 维度补充社交倾向**：

- **E 型**：添加"社交倾向：外向型，从外部世界获取能量"
- **I 型**：添加"社交倾向：内向型，需要独处恢复能量"

### Layer 2：说话风格层

**根据 S/N 维度调整表达内容**：

| 维度 | 表达特点 | 建议补充 |
|------|---------|---------|
| S 型 | 关注具体细节、实际经验 | "表达偏好：喜欢用具体例子说明，关注可操作的细节" |
| N 型 | 关注整体模式、抽象概念 | "表达偏好：喜欢讨论概念和可能性，关注宏观图景" |

**根据 T/F 维度调整表达方式**：

| 维度 | 表达特点 | 建议补充 |
|------|---------|---------|
| T 型 | 直接、逻辑性强 | "沟通风格：直接坦诚，重视逻辑一致性" |
| F 型 | 温和、注重感受 | "沟通风格：委婉体贴，重视人际和谐" |

**根据 J/P 维度调整表达结构**：

| 维度 | 表达特点 | 建议补充 |
|------|---------|---------|
| J 型 | 有条理、明确结论 | "表达结构：喜欢给出明确结论，信息组织有序" |
| P 型 | 开放、探讨可能 | "表达结构：喜欢探讨多种可能性，保持开放态度" |

### Layer 3：情感与决策模式层

**这是 MBTI 影响最直接的层级**

**T/F 维度 → 决策偏好**：

```markdown
### 决策模式
- 理性 vs 感性：{T: 偏向理性，依据逻辑分析做决定 / F: 偏向感性，考虑他人感受做决定}
- 决策依据：{T: 客观事实和逻辑推演 / F: 价值观和人际关系}
- 冲突处理：{T: 倾向于直接指出问题 / F: 倾向于维护和谐}
```

**J/P 维度 → 计划偏好**：

```markdown
### 计划与执行
- 计划 vs 随性：{J: 偏向计划，喜欢提前安排 / P: 偏向随性，喜欢保持灵活}
- 截止日期态度：{J: 重视截止日期，提前完成 / P: 压力下工作，灵活应对}
- 任务处理：{J: 专注完成一项再开始下一项 / P: 可能同时进行多项任务}
```

**情感触发器（根据 MBTI 类型推断）**：

| 类型 | 可能的情感触发器 |
|------|-----------------|
| TJ 型 | 效率低下、逻辑混乱、规则被打破 |
| TP 型 | 被限制自由、重复性工作、缺乏创新 |
| FJ 型 | 人际冲突、被忽视感受、团队不和谐 |
| FP 型 | 价值观被挑战、缺乏真诚、被迫做不喜欢的事 |

### Layer 4：人际行为层

**E/I 维度直接影响**：

```markdown
### 社交模式
- 能量来源：{E: 从社交互动中获取能量 / I: 从独处中恢复能量}
- 社交偏好：{E: 喜欢广泛的社交圈 / I: 偏好深度的小圈子}
- 互动方式：{E: 主动发起，先说后想 / I: 被动等待，先想后说}
```

**T/F 维度影响人际互动**：

```markdown
### 人际互动风格
- 冲突态度：{T: 不回避冲突，就事论事 / F: 回避冲突，优先和谐}
- 表达关心：{T: 通过实际行动解决问题 / F: 通过情感支持和倾听}
- 接受批评：{T: 理性接受，关注改进 / F: 可能受伤，需要温和方式}
```

## 16 种 MBTI 类型的 Persona 特征速查

### 分析家组 (NT)

**INTJ（建筑师）**
- Layer 2：表达简洁直接，喜欢深入讨论
- Layer 3：高度理性，长期规划者
- Layer 4：社交圈小但深度，独立性强

**INTP（逻辑学家）**
- Layer 2：表达抽象，喜欢理论探讨
- Layer 3：分析型决策，灵活应对
- Layer 4：社交需求低，享受智力交流

**ENTJ（指挥官）**
- Layer 2：表达有力，目标导向
- Layer 3：果断决策，高效执行
- Layer 4：社交活跃，喜欢领导角色

**ENTP（辩论家）**
- Layer 2：表达机智，喜欢辩论
- Layer 3：创新决策，适应变化
- Layer 4：社交广泛，享受智力碰撞

### 外交家组 (NF)

**INFJ（提倡者）**
- Layer 2：表达温和有深度
- Layer 3：价值观驱动，有洞察力
- Layer 4：深度社交，善于倾听

**INFP（调停者）**
- Layer 2：表达富有诗意和想象力
- Layer 3：理想主义，追求意义
- Layer 4：小圈子深度社交，真诚敏感

**ENFJ（主人公）**
- Layer 2：表达有感染力和说服力
- Layer 3：考虑他人，善于协调
- Layer 4：社交活跃，喜欢帮助他人

**ENFP（竞选者）**
- Layer 2：表达热情洋溢，富有创意
- Layer 3：灵活决策，追求可能
- Layer 4：社交广泛，善于激励他人

### 守护者组 (SJ)

**ISTJ（物流师）**
- Layer 2：表达准确，注重事实
- Layer 3：依据规则和经验决策
- Layer 4：社交稳定，忠诚可靠

**ISFJ（守卫者）**
- Layer 2：表达温和，注重细节
- Layer 3：考虑他人需求做决定
- Layer 4：深度社交，乐于付出

**ESTJ（总经理）**
- Layer 2：表达直接，有条理
- Layer 3：依据规则和效率决策
- Layer 4：社交活跃，喜欢组织

**ESFJ（执政官）**
- Layer 2：表达热情，关注他人
- Layer 3：维护和谐，考虑他人
- Layer 4：社交活跃，善于照顾他人

### 探险家组 (SP)

**ISTP（鉴赏家）**
- Layer 2：表达简洁，行动导向
- Layer 3：灵活决策，实用主义
- Layer 4：社交适度，享受独立

**ISFP（探险家）**
- Layer 2：表达温和，有艺术感
- Layer 3：依据价值观和感受决策
- Layer 4：小圈子社交，注重体验

**ESTP（企业家）**
- Layer 2：表达直接，充满活力
- Layer 3：快速决策，注重当下
- Layer 4：社交活跃，喜欢刺激

**ESFP（表演者）**
- Layer 2：表达热情，富有感染力
- Layer 3：灵活决策，享受过程
- Layer 4：社交广泛，喜欢成为焦点

```markdown

## 新增内容

### Layer 1：身份层
[新增的 MBTI 相关身份信息]

### Layer 2：说话风格层
[根据 MBTI 补充的表达风格]

### Layer 3：情感与决策模式层
[根据 MBTI 补充的决策模式]

### Layer 4：人际行为层
[根据 MBTI 补充的人际行为模式]

## 一致性检查

[对比 MBTI 结果与现有 persona.md 的内容，指出一致之处和可能的差异]

## 建议

[对用户的具体建议，如是否需要调整某些内容]
```

---

## 注意事项

1. **不覆盖已有内容**：MBTI 结果作为补充，不替换 persona.md 中已有的事实性描述
2. **保持一致性**：如果 MBTI 结果与聊天记录分析结果有显著差异，需要指出并询问用户
3. **避免刻板印象**：MBTI 是倾向性指标，不是绝对分类，避免过度泛化
4. **尊重用户自我认知**：如果用户对自己的描述与 MBTI 结果不符，以用户描述为准
5. **标注来源**：新增内容应标注来源为"MBTI 测试"，便于后续追溯

## 示例

### 输入

```
MBTI 类型：INTJ（建筑师）
现有 persona.md：
- 身份层：程序员，30岁，北京
- 说话风格：喜欢用技术术语，表达简洁
- 决策模式：理性分析为主
```

### 输出

```markdown
# Persona 更新建议

## 新增内容

### Layer 1：身份层
- MBTI：INTJ（建筑师）
- 社交倾向：内向型，需要独处恢复能量

### Layer 2：说话风格层
- 表达偏好：喜欢讨论概念和可能性，关注宏观图景（N 型特征）
- 沟通风格：直接坦诚，重视逻辑一致性（T 型特征）
- 表达结构：喜欢给出明确结论，信息组织有序（J 型特征）

### Layer 3：情感与决策模式层
- 决策模式：偏向理性，依据逻辑分析做决定
- 决策依据：客观事实和逻辑推演
- 计划与执行：偏向计划，喜欢提前安排
- 情感触发器：效率低下、逻辑混乱、规则被打破

### Layer 4：人际行为层
- 社交模式：从独处中恢复能量，偏好深度的小圈子
- 互动方式：先想后说，被动等待互动
- 冲突态度：不回避冲突，就事论事

## 一致性检查

✅ MBTI 结果与现有 persona 高度一致：
- INTJ 的理性分析与"理性分析为主"一致
- INTJ 的简洁表达与"表达简洁"一致
- 程序员职业与 INTJ 的常见职业倾向一致

## 建议

1. MBTI 结果与现有 persona 高度吻合，可直接整合
2. 建议补充 INTJ 的潜在盲点：可能显得傲慢、过于挑剔
3. 建议补充情感表达模式：INTJ 通常不善于表达情感，但这不代表没有情感
```
